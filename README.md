# School Announcement and Event Management System

A Laravel-based web application developed as a mini-capstone project.  
The system allows school administrators to post announcements and manage events, while students can view updates in real time.

## Features
- User authentication (Admin)
- Announcement management (CRUD)
- Event management (CRUD)
- Responsive user interface

## Technologies Used
- Laravel
- PHP
- MySQL
- Blade Templates
- Bootstrap / Tailwind CSS

## Installation
1. Clone the repository
2. Run `composer install`
3. Copy `.env.example` to `.env`
4. Configure database in `.env`
5. Run `php artisan migrate`
6. Run `php artisan serve`
